# Project

This repository contains a React + Vite dashboard application.

Quick start:

```powershell
npm install
npm run dev -- --host
```

Build for production:

```powershell
npm run build
npm run preview
```

Tech stack:

- Vite
- React
- TypeScript
- Tailwind CSS
